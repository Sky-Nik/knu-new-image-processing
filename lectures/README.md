## Лекції з обробки та розпізнавання зображень

1. Оглядова лекція. Структура курсу. Налаштування середовища. Огляд можливостей існуючих бібліотек. [Jupyter Notebook](1-Introduction.ipynb)

2. Поняття ядра, згортки. Гаусове згладження. Градієнтний оператор Собеля. Детектор границь Кенні. Огляд 1-ої лаборатороної роботи. [Jupyter Notebook](2-Edge-Detection.ipynb)

3. Оператор порогування. Евристики вибору порогу. Локальне порогування. Алгоритми знаходження порогу. Алгоритм Оцу. [Jupyter Notebook](3-Otsu-Thresholding.ipynb)

6. Каскадний класифікатор. Застосування до розпізнавання. Розпізнавання облич. Огляд 3-ої лаборатороної роботи. [Jupyter Notebook](6-Face-Recognition.ipynb)
